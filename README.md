# NadiMovies
 
## This is a clean architecture app example built with :
* Clean Code
* Unit Test
* MVVM 
* ~~Databinding~~ -> ViewBinding
* ~~LiveData~~ -> StateFlow
* ViewModel 
* Coroutines 
* Retrofit
* Room (Offline Caching)
* Dagger Hilt
* Mockito
* Memory Leak (Leakcanary)
* Viewpager2
* Youtube Player
* Extension Functions
* Repository Pattern
* Navigation Component
